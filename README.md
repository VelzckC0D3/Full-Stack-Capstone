## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

Node.js
Visual Studio Code
Browser

### Setup

git clone https://github.com/VelzckC0D3/Land-Rover-E-Commerce.git

cd Land-Rover-E-Commerce

### Install

npm install

### Usage

npm run start

### Run tests

npm test

### Deployment

npm run build
